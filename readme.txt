== Description ==
Build custom page layouts using Advanced Custom Field\'s \"Flexible Content\" field type.

Current sections include:

* 1 Column
* 2 Column
* 3 Column
* Full Width Image
* 2 Images
* 3 Images
* Image and Content
* Highlighted Section
* YouTube Video
